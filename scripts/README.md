# Stubs de configuración para VMs de producción

Scripts de PowerShell para configurar las VMs Windows del entorno productivo.

| VM | IP | Script |
|---|---|---|
| SQL Server | `192.168.1.102` | `stub-sqlserver-vm.ps1` |
| Active Directory | `192.168.1.10` | `stub-ad-vm.ps1` |

Ejecutar en PowerShell como Administrador en la VM correspondiente.
